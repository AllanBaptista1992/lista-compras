# 🛒 Lista de Compras

Aplicação desenvolvida com **Python**, **Streamlit**, **SQLite** e **Plotly** para gerenciar compras e acompanhar gastos de forma simples e intuitiva.

## 🚀 Funcionalidades

* Adicionar produtos
* Editar produtos
* Excluir produtos
* Calcular total da compra automaticamente
* Histórico de compras
* Dashboard com gráficos
* Gastos diários, mensais e anuais
* Tema claro e escuro
* Layout responsivo para desktop, tablet e celular
* Armazenamento local com SQLite

## 🛠 Tecnologias

* Python
* Streamlit
* SQLite
* Pandas
* Plotly

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/AllanBaptista1992/lista-compras.git
```

Entre na pasta do projeto:

```bash
cd lista-compras
```

Crie o ambiente virtual:

```bash
python -m venv venv
```

Ative o ambiente virtual:

```bash
.\venv\Scripts\Activate
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute a aplicação:

```bash
streamlit run app.py
```

* Gastos do dia
* Gastos do mês

* Gráfico de gastos por mês
